# 热搜详情与图片优化设计

## 目标

让妈妈可以点击微博热搜进入实时讨论，并在日报中展开查看不编造的说明；让新闻配图最多两张、经过压缩且不在 GitHub 运行器中长期累积。

## 热搜

NewsNow 提供的是热词和微博搜索链接，不含事件正文。每个热词保留标题、链接和来源字段；网页用 `details` 作为二级菜单，摘要固定说明为“点击查看微博实时讨论”，不将关键词虚构为新闻事实。企业微信使用可点击的 Markdown 链接。

## 图片

从已提取文章页面收集最多两张候选图片，跳过中新网通用 RSS 图。图片下载到本次 GitHub Actions 的 `news/images` 目录，以缩放和 JPEG/WebP 压缩后作为当天静态页资源；运行器每次全新启动，Pages 发布也只使用本次产物，不会长期累积。单张最大约 300KB；没有可靠图片则不显示。

## 验证

测试热搜链接和二级菜单、最多两张图、图片压缩后使用相对路径，以及旧测试全量通过。
